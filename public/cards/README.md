# Card Image Assets

Put all card face and back images in this folder.

## Naming format

Use PNG files and match these names exactly:

- Fronts: `<rank>_of_<suit>.png`
- Back: `back.png`

Examples:

- `ace_of_hearts.png`
- `10_of_spades.png`
- `queen_of_clubs.png`
- `2_of_diamonds.png`
- `back.png` = card back image

## Supported ranks

`2 3 4 5 6 7 8 9 10 jack queen king ace`

## Supported suits

`hearts diamonds clubs spades`

The UI falls back to text labels if an image file is missing.
